# Identity and Access Management

Usaremos
  - Roles IAM
  - MFA (Multi-Factor Authentication)

1. Crear persona
2. (check) Proporcione acceso de usuario a la consola de administración de AWS: opcional
3. Agregamos permisos directamente, AdministratorAccess para comenzar

#### Ver el alias de la cuenta root
  IAM > Dashboard > Account Alias (Parte derecha)
Se inicia sesion con el id o alias de la cuenta + el nuevo usuario y la contraseña
